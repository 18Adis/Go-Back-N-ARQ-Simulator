# Go-Back-N ARQ Simulator (Python + Tkinter)

This project is a visual simulator for the Go-Back-N ARQ protocol implemented in Python using Tkinter.

## Features

- Visual sender and receiver timelines
- Configurable:
  - Total number of frames
  - Window size
  - Manually specify lost frames
- Animations for:
  - Frame transmission
  - Lost frames (simulated)
  - Cumulative ACKs
- Option to record the whole simulation and save it as an animated GIF

## Requirements

- Python 3.x
- Pillow
